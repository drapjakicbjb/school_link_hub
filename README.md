# 🏫 Dr. A.P.J. Abdul Kalam School Link Hub

A premium, interactive web portal designed for the students and faculty of **Dr. A.P.J. Abdul Kalam Inter College**. This platform serves as a central "Digital Atrium," providing quick, organized access to all essential educational resources and communication channels.

---

## ✨ Key Features

- **Centralized Resource Hub**: Quick links to WhatsApp Support, YouTube Channels, Official Website, and specialized Study Hubs.
- **"Coast & Dock" Animation**: A custom-engineered JavaScript scroll engine that detaches branding logos from the navbar and smoothly docks them into the footer, ensuring content remains unobstructed during browsing.
- **Glassmorphic UI**: High-end modern design using vibrant gradients, ambient shadows, and blur effects for a premium institutional feel.
- **Dynamic Theme Support**: A floating action button allows users to toggle seamlessly between **Light** and **Dark** modes.
- **Fully Responsive**: Optimized for every device, from wide desktops to mobile phones, with a stacked layout for smaller screens.
- **High Performance**: Built with pure Vanilla JS and CSS for lightning-fast load times and smooth 60fps animations.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS3**: Modern styling including CSS Variables, Flexbox/Grid, and hardware-accelerated animations.
- **ES6+ JavaScript**: Custom physics-based interpolation engine for scroll animations and dynamic card rendering.
- **Google Fonts**: Inter & Outfit for modern, readable typography.
- **FontAwesome**: High-quality vector iconography.

## 🚀 How to Run Locally

Since this is a static frontend application, you can run it without any complex setup:

1. **Clone or Download** the repository to your local machine.
2. Ensure the branding assets (`rsy_logo.png`, `new_logo.png`, `dal_sringari_logo-removebg-preview.png`) are in the root directory.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, or Edge).

---

## 📖 Project Structure

- `index.html`: The main entry point and structure of the portal.
- `style.css`: The "Design System" containing all colors, tokens, and layouts.
- `script.js`: The "Logic Engine" handling theme switching, search, and the logo animation engine.
- `*.png`: Institutional branding and logo assets.
